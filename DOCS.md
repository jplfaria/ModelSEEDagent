## 🏃‍♂️ Quick Start

```bash
# Install ModelSEEDagent
pip install -e .

# View help
modelseed-agent --help

# Set up LLM backend (Argo Gateway recommended)
modelseed-agent setup --backend argo

# Start interactive analysis
modelseed-agent interactive
```
